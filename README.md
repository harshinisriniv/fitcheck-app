# FitCheck

FitCheck is a mobile app that helps users share and explore fashion outfits with tagged details. Users can view posts, save inspiration, and discover styles from the community.

## Features

- View detailed posts with images and tags
- Save favorite posts to your personal inspiration collection
- Browse other users’ outfits
- User authentication and profile management
- Responsive and intuitive UI built with React Native and Expo
- Firebase Firestore backend for data storage and authentication
